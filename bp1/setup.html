Supabase Setup

Configure your Supabase credentials to activate the website.

Current Configuration:

URL: Not set

Key: Not set

How to get your credentials:

  1. 1. Go to supabase.com and create a project
  2. 2. Go to Settings → API in your project dashboard
  3. 3. Copy the "Project URL" and "anon public" key
  4. 4. Run the SQL from supabase/schema.sql in SQL Editor
  5. 5. Enter your credentials above and save

How to verify access:

  1. 1. After configuring credentials, go to purchase.html
  2. 2. Register/login and submit a payment
  3. 3. Manually update users_access.paid_access = true in Supabase
  4. 4. User can then login, enter PIN 055183, and access course

Payment Flow

After user submits payment proof, you manually approve in Supabase dashboard by setting paid_access = true in the users_access table.
