Upload Video Instructions

To add your videos:

  1. Name your video file according to the module: module-1.mp4, module-2.mp4, etc.
  2. Create a folder named videos in your project root
  3. Upload your video files to the videos/ folder
  4. Open course.html and find the openVideo() function
  5. Update the video source paths in the function (example below)

How to update video paths:

// Find this function in the script below and update:
function openVideo(card, moduleId) {
    const videoSources = {
        'MODULE_1': 'videos/module-1.mp4',
        'MODULE_2': 'videos/module-2.mp4',
        'MODULE_3': 'videos/module-3.mp4',
        'MODULE_4': 'videos/module-4.mp4',
        'MODULE_5': 'videos/module-5.mp4'
    };
    // ... rest of function
}

Supported formats: MP4, WebM, MOV
